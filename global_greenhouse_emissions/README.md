# EDGAR - Emissions Database for Global Atmospheric Research (EDGAR) Datasets

The files in this directory contain EDGAR emissions data for the years 1970 through 2023. The original files can be access here: https://data.jrc.ec.europa.eu/dataset/88c4dde4-05e0-40cd-a5b9-19d536f1791a

The data is provided in XLSX format. The details of the data are as follows:

* IEA_EDGAR_CO2_1970_2023.xlsx - CO2 emissions data for the years 1970 through 2023. The data is broken down by country and sector. The original "readme" file is included as co2_readme.html
* EDGAR_N2O_1970_2023.xlsx - N2O emissions data for the years 1970 through 2023. The data is broken down by country and sector. The original "readme" file is included as n20_readme.html
* EDGAR_CH4_1970_2023.xlsx - CH4 emissions data for the years 1970 through 2023. The data is broken down by country and sector. The original "readme" file is included as ch4_readme.html
* EDGAR_AR5_GHG_1970_2023.xlsx - Combined Greenhouse gas emissions data for the years 1970 through 2023. The data is broken down by country and sector. The original "readme" file is included as ghg_readme.html
    - Note: The values in this dataset are the combined greenhouse gas emissions in CO2 equivalent units and include fluorinated gases, which we have not included in this directory, since that data does not go back to 1970.

The data is provided by the European Commission's Joint Research Centre (JRC). The data is provided as-is and the original source should be consulted for the most up-to-date information.
